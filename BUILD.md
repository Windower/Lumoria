## Dependencies

Arch BTW:

```sh
sudo pacman -S vala meson gtk4 libadwaita glib2 json-glib libsoup3 libgee libarchive libmspack vte4
```

Flatpak SDK:

```sh
flatpak remote-add --if-not-exists --user flathub https://dl.flathub.org/repo/flathub.flatpakrepo
flatpak install --user -y flathub org.flatpak.Builder org.gnome.Sdk//50 org.gnome.Platform//50 org.freedesktop.Sdk.Extension.vala//25.08
```

## Development

Host build:

```sh
make host
make host-run
```

Flatpak build:

```sh
make flatpak.dev.build
make flatpak.dev.run
```

## Release Testing Builds

Create a `.env` in the project root with the variables used by the Makefile release targets.

Then:

```sh
make release.signed
```

## Notes

- `lumoria.flatpakrepo` is generated by make. Don't hand-edit it.
